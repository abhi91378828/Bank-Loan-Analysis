# Bank Loan Analysis Dashboard

## Project Overview

This project analyzes bank loan data using SQL Server and Tableau to identify loan performance, borrower characteristics, and lending trends.

## Tools Used

* SQL Server
* SQL Server Management Studio (SSMS)
* Tableau
* Excel / CSV

## Key KPIs

* Total Loan Applications
* Total Funded Amount
* Total Amount Received
* Average Interest Rate
* Average Debt-to-Income Ratio (DTI)

## Analysis Performed

* Good Loan vs Bad Loan Analysis
* Loan Status Analysis
* State-wise Loan Distribution
* Term Analysis
* Home Ownership Analysis
* Purpose Analysis
* Monthly Loan Trends

## Files Included

* Tableau Dashboard (.twbx)
* SQL Queries (.sql)
* Dataset (.csv)
* Dashboard Screenshots
